### EduTube — Distraction-Free Learning

`React` `Node.js` `Tailwind CSS` `Firebase` `Gemini API`

Full-stack platform for managing YouTube playlists inside a focused, recommendation-free embedded player — built with a Node.js backend exposing **12 REST API routes**.

- Playlist categorization, lecture scheduling & note persistence
- Pomodoro timer, to-do list, and lecture planner
- **Google Gemini** integration for real-time doubt resolution, quiz generation, and AI note summaries
- Persistent cross-session state via Firebase Firestore

[![Live Demo](https://img.shields.io/badge/Live_Demo-7c3aed?style=for-the-badge&logo=vercel&logoColor=white)](https://edutube1404.vercel.app/)

### VendorHub — Local Vendor Marketplace

`Java` `Spring Boot` `MySQL` `REST API`

Full-stack MVP marketplace for **50+ local vendors**, built in Java and Spring Boot with **8 RESTful API endpoints** and a normalized MySQL schema across 6 tables.

- OAuth authentication & structured error handling
- Text-to-speech order notification module for low-literacy vendors
- Normalized DB schema: vendors, products, orders, users, sessions, notifications
- Delivered as backend lead in a **36-hour hackathon window**

> 🥈 Institute-Level Finalist — Smart India Hackathon 2023

</td>
</tr>

### AI Image Detector — Hybrid Deep Learning Model

`Python` `PyTorch` `OpenCV`

Hybrid classification pipeline that distinguishes real vs. AI-generated images, outperforming single-architecture baselines on benchmark datasets.

- Fused **Swin Transformer + MobileNetV3** via gated attention mechanism
- Patch extraction and test-time augmentation (TTA) to reduce false positives
- Evaluated on benchmark datasets for detection accuracy
